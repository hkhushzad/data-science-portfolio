# Data Science Portfolio

Welcome to my **Data Science Portfolio**, showcasing projects using **SQL, R, and Python**. These projects demonstrate skills in **data cleaning, analysis, visualization, and modeling**, with reproducible workflows.

I am a **Master’s student in Data Science** (finishing July 2026) with a background in **Biological Sciences** and **5 years of experience as a Pharmacy Technician**, applying analytical skills to real-world datasets.

---

## Projects Overview

| Project | Description | Language / Tool | File |
|---------|-------------|----------------|------|
| Banking Relational Database | Designed and implemented a normalized banking database schema for customers, accounts, transactions, and branches. | SQL | [banking_relational_database.sql](banking_relational_database.sql) |
| Bitcoin Transactions Analysis | Analyzed Bitcoin transactions to extract insights on volume, frequency, and temporal patterns. | SQL | [bitcoin_transactions_analysis.sql](bitcoin_transactions_analysis.sql) |
| CDC Public Health Analysis | Explored BRFSS 2021 dataset to analyze fruit consumption, general health, and checkup patterns; included regression modeling. | R | [cdc_public_health_analysis.R](cdc_public_health_analysis.R) |
| German Credit Regression | Predicted Credit Amount from Age and Duration using linear regression, showcasing Python data analysis and modeling. | Python | [german_credit_regression.py](german_credit_regression.py) |

## Key Skills Demonstrated
- **SQL:** Database design, normalization, DDL statements, complex queries, aggregate/window functions, query optimization  
- **R:** Data cleaning, exploratory data analysis (EDA), descriptive statistics, regression modeling, reproducible workflow  
- **Python:** Data cleaning with pandas, linear regression with statsmodels, scripting, reproducible analysis  
- **Data Science Foundations:** Data preprocessing, statistical analysis, model evaluation, visualization, and reporting

## How to Use This Portfolio
1. Click on each file to explore **SQL, R, or Python code**.  
2. Run scripts locally following the file instructions.  
   - For Python/R projects, ensure the dataset files (e.g., `brfss2021.csv` or `german_credit_data.csv`) are in the same folder.  
3. Review code comments for step-by-step explanations of workflow and analysis.
